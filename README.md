# ­ЪЊЊ Phone Note App

рЃћрЃА рЃљрЃарЃўрЃА Kivy-рЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў рЃАрЃбрЃўрЃџрЃўрЃА рЃюрЃЮрЃБрЃЌрЃћрЃЉрЃўрЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ, рЃюрЃљрЃ«рЃЮ рЃЊрЃљ рЃгрЃљрЃерЃљрЃџрЃЮ рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃћрЃЉрЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃЊрЃљ рЃАрЃгрЃарЃљрЃцрЃљрЃЊ.

---

## ­ЪЊ▒ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃў:

- ­Ъћљ рЃърЃљрЃарЃЮрЃџрЃўрЃЌ рЃЊрЃљрЃфрЃБрЃџрЃў рЃерЃћрЃАрЃЋрЃџрЃљ
- ­ЪДа рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃџрЃљрЃЏрЃљрЃќрЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў
- РюЇ№ИЈ рЃюрЃЮрЃБрЃЌрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ (рЃАрЃљрЃЌрЃљрЃБрЃарЃўрЃЌ рЃЊрЃљ рЃљрЃдрЃгрЃћрЃарЃўрЃЌ)
- ­ЪЊІ рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃўрЃА рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃў рЃАрЃўрЃљ
- РЮї рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃўрЃА рЃгрЃљрЃерЃџрЃљ
- ­ЪД╝ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ SQLite рЃЉрЃљрЃќрЃљрЃерЃў

## ­Ъџђ рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ:

1. рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ:

2. рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃљ:
```bash

```

3. рЃЉрЃљрЃќрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ:
рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃљрЃ«рЃљрЃџрЃў рЃцрЃљрЃўрЃџрЃў рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ sqlite.db

4. рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ main.py:
```bash

```
